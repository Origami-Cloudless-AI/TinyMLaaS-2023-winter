TinyMLaaS installation guide (Linux)

Prerequisites

  • Python3
  • Docker
  • Nbdev

Installation

Web App

  1. Clone the GitHub repository:
    git clone git@github.com:Origami-TinyML/tflm_hello_world.git
  1. Run (might require sudo):
    nbdev_install
  1. Install docker-compose:
    sudo apt-get install docker-compose

  1. Fill the missing environment variables in the docker-compose.yml file: >REMEMBER: DO NOT COMMIT ANY SENSITIVE INFORMATION TO THE REPOSITORY.

  2. Launch the app (sudo password is required):

    make app

Open localhost:8501 on your web browser to start using the TinyMLaaS app.

Relay Server

Choose a relay server (has only been tested on Cubli).

  1. Clone the GitHub repository:
    git clone git@github.com:Origami-TinyML/tflm_hello_world.git
  1. Run (might require sudo):
    nbdev_install
  1. Check the relay servers’ DOCKERHUB_USER variable and modify it if necessary:
    head relay/components/install.py
  1. Run the relay server (sudo might be required):
    python3 relay/main.py
